# Central Achat Publique

Bootstrap d'un monolithe modulaire (FastAPI + SQLAlchemy + Alembic) avec regles strictes de migrations.

## Demarrage rapide
```bash
./scripts/start.sh
```

## Commandes utiles
```bash
./scripts/db_migrate.sh
./scripts/db_reset.sh
./scripts/test.sh
./scripts/format.sh
```

## Pre-requis
- Python 3.11+
- Docker (pour PostgreSQL)
